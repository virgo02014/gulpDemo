1. 样式放在 `/css/*.css` 目录下
2. `npm install` 安装依赖
3. 执行 `npm start` 会自动合并、自动补全
4. `/dist/main.css` 为最后生成文件


```
.
angular-topic
├── css                 //样式文件
├── dist                //自动补全+合并后存放的样式文件
├── images              //图片
├── node_modules        //package.json
├── index.html          //首页
├── package.json        //npm配置文件
├── .gitignore          //git配置文件
└── README.md           //项目的基础文档
.
```
